Open Slack workspace

What you need for a real Slack setup

  • A Slack bot token xoxb-... stored only in backend secrets or Supabase Edge Function environment variables.
  • Your Slack channel ID C... for the channel that should receive updates.
  • The Slack signing secret if you want slash commands or interactive replies.
  • A public relay endpoint URL that this website can call safely.

What goes in this website

  • window.SLACK_WEBHOOK_PROXY_URL — your public relay endpoint that forwards approved events to Slack.
  • window.SLACK_WORKSPACE_URL — optional direct link to open your Slack workspace.
  • window.SLACK_DEFAULT_CHANNEL — optional label so the admin page shows which channel receives updates.

Do not paste raw Slack bot tokens, webhook secrets, or signing secrets into this frontend config file. Those belong in a private backend.
